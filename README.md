# Homelabinator

## What is homelabinator?
Homelabinator is an easy way to setup your homelab end to end without having to jump through the flaming hoops of networking, containerization etc.!

## How is the frontend built?
Homelabinator is built on the shoulders of giants, using DaisyUI and Tailwind CSS.

## Curious on how the backend works?
Check out the [backend repo](https://github.com/homelabinator/homelabinator-backend) where ISOs are built with Nix.

## Don't trust us?
Our codebase is public! Feel free to look around and open issues or PRs as you see fit.
